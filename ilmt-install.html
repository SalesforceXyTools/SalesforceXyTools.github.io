IBM License Metric Tool 7.5安装设定 ILMT

IBM License Metric Tool 7.5安装设定

安装ILMT比较繁琐，大致顺序如下

DB2 v9.7.0.9安装->

ILMT V7.5.0.0->

TIP2.2.0.0更新到2.2.0.3->

TIP2.2.0.3更新到2.2.0.7->

ILMT V7.5Fixpack1安装->

ILMTV7.5Interim Fix 9安装

 

硬件配置

安装的时候，
如果1G内存，512M paging space，会报内存不足
4Gn内存,512M PS 不会错误，大概需要1h内
1G内存，4.5G ps，也不会出错，不过时间会要3.5h以上
另外，SoftwareCatalog更新，一般要24h小时内。

 

安装顺序：

1.下载ILMT安装包

Download IBM License Metric Tool 7.5 for AIX platforms

Download IBM License Metric Tool 7.2.2 for AIX platforms

2.下载DB2 V9.7版本

Download DB2 Fix Packs by version for DB2 for Linux, UNIX and Windows

3.安装DB2 9.7

4.ILMT安装前提

10.
升级TIP2.2.0.0->2.2.0.3，[参考](http://pic.dhe.ibm.com/infocenter/tivihelp/v53r1/index.jsp?topic=%2Fcom.ibm.lmt75.doc%2Fcom.ibm.license.mgmt.planinconf.doc%2Ft_updating_server_silently.html)

[shell]
./patch_installer-aix-ppc64.sh -silent -patchHome `pwd` -response `pwd`/tip-update.txt
vi tip-update.txt
./patch_installer-aix-ppc64.sh -patchHome `pwd` -response `pwd`/tip-update.txt -silent
[/shell]

升级TIP2.2.0.3->2.2.0.7

[shell]
#./install.sh -i silent -f response.txt
[/shell]
