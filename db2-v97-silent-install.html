DB2 V9.7静默安装

DB2 V9.7静默安装

1.创建db2ese.rsp静默安装文件

[php]

LIC_AGREEMENT = ACCEPT
PROD = ENTERPRISE_SERVER_EDITION
FILE = /opt/IBM/db2/V9.7
INSTALL_TYPE = TYPICAL
DAS_CONTACT_LIST = LOCAL
DAS_USERNAME = dasusr1
DAS_GROUP_NAME = dasadm1
DAS_HOME_DIRECTORY = /home/dasusr1
DAS_PASSWORD = dasusr1
INSTANCE = db2inst1
db2inst1.TYPE = ese
db2inst1.NAME = db2inst1
db2inst1.GROUP_NAME = db2grp1
db2inst1.HOME_DIRECTORY = /home/db2inst1
db2inst1.PASSWORD = db2inst1
db2inst1.AUTOSTART = YES
db2inst1.SVCENAME = db2c_db2inst1
db2inst1.PORT_NUMBER = 50000
db2inst1.FCM_PORT_NUMBER = 60000
db2inst1.MAX_LOGICAL_NODES = 4
db2inst1.CONFIGURE_TEXT_SEARCH = NO
db2inst1.FENCED_USERNAME = db2fenc1
db2inst1.FENCED_GROUP_NAME = db2fgrp1
db2inst1.FENCED_HOME_DIRECTORY = /home/db2fenc1
db2inst1.FENCED_PASSWORD = db2fenc1
LANG = EN
INSTALL_TSAMP = NO
[/php]

2.DB2 V9.7安装指令

[shell]

#rm　-r /opt/IBM/db2/V9.7/lost+found

#cd　工作目录

#./db2setup -r /work/db2ese.rsp
DBI1191I db2setup is installing and configuring DB2 according to the
response file provided. Please wait.

iocp0 Filesystem size changed to 3276800
The execution completed successfully.

For more information see the DB2 installation log at "/tmp/db2setup.log".

[/shell]

3.语言包静默安装文件db2nlpack.rsp

[shell]

PROD = NATIONAL_LANGUAGE_PACKAGE
FILE = /opt/IBM/db2/V9.7
INSTALL_TYPE = CUSTOM
LANG = EN
LANG = JP
[/shell]

4.语言包安装前，需要停止DB2

[shell]

#su - db2inst1
$db2 force application all
DB20000I The FORCE APPLICATION command completed successfully.
DB21024I This command is asynchronous and may not be effective immediately.

$db2 terminate
DB20000I The TERMINATE command completed successfully.
$db2stop
2014-05-20 16:35:44 0 0 SQL1064N DB2STOP processing was successful.
SQL1064N DB2STOP processing was successful.
$exit

#su - dasusr1
$db2admin stop
SQL4407W The DB2 Administration Server was stopped successfully.
$exit
#
[/shell]

5.语言包安装

[shell]

./db2setup -r db2nlpack.rsp

DBI1191I db2setup is installing and configuring DB2 according to the
response file provided. Please wait.

The response file specified "/tmp/GD_P/DB2/nlpack/db2nlpack.rsp" is not
valid.

For more information see the DB2 installation log at "/tmp/db2setup.log".

#/opt/IBM/db2/V9.7/install/db2ls -q -b /opt/IBM/db2V9.7/ -a | grep -i message
DB2_PRODUCT_MESSAGES_EN 9.7.0.9 9 Product Messages - English
DB2_PRODUCT_MESSAGES_JP 9.7.0.9 9 Product Messages - Japanese
#
[/shell]

[shell]

[/shell]
