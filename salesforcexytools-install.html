Prerequisites

  • Sublime Text 3 http://www.sublimetext.com/3
  • Sublime Text Package Control https://packagecontrol.io/installation

Plugin Installation(Use Package Control to Install)

  1. Open Sublime Text 3

  2. Run Package Control: Install Package command

    • Running commands from Sublime Text

  3. Search for SalesforceXyTools

  4. Hit Enter, That is all.
